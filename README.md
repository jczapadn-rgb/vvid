
Цель работы
Разработать модель машинного обучения для прогнозирования прочности бетона на сжатие на основе состава смеси и возраста образцов.

Выбранная модель
Модель: Random Forest Regressor (Случайный лес)

Почему выбрана эта модель:

Устойчива к выбросам и шуму в данных

Позволяет оценить важность каждого компонента бетона

Не требует масштабирования признаков

Хорошо работает с нелинейными зависимостями

Даёт интерпретируемые результаты для инженерных задач

Параметры модели:

Количество деревьев: 100

Остальные параметры — по умолчанию

Анализ данных перед обучением
Перед обучением модели был проведён следующий анализ:

Первичный осмотр данных:

Загружено 1030 образцов бетона

8 признаков (состав смеси + возраст)

Целевая переменная — прочность на сжатие (МПа)

Корреляционный анализ:

Построена матрица корреляции Пирсона

Выявлены ключевые зависимости:

Цемент — сильная положительная корреляция (~0.5)

Возраст — умеренная положительная корреляция (~0.33)

Вода — отрицательная корреляция (~ -0.29)

Разделение данных:

Обучающая выборка: 80% (824 образца)

Тестовая выборка: 20% (206 образцов)

Визуальный анализ:

Построены гистограммы распределений

Визуализирована важность компонентов

Выводы по метрикам модели
Метрика	Обучающая выборка	Тестовая выборка
MAE (средняя абсолютная ошибка)	~2.7 МПа	~4.7 МПа
RMSE (среднеквадратичная ошибка)	~4.1 МПа	~6.2 МПа
R² (коэффициент детерминации)	~0.97	~0.92
Ключевые выводы:

Высокое качество предсказаний:
Модель объясняет 92% вариативности прочности бетона на тестовых данных (R² = 0.92)

Приемлемая точность для инженерии:
Средняя ошибка предсказания составляет ±4.7 МПа при диапазоне прочности 15–80 МПа (относительная ошибка ~10-15%)

Небольшое переобучение:
Разрыв R² между обучением (0.97) и тестом (0.92) составляет 0.05 — переобучение незначительное, модель обобщает хорошо

Наиболее важные компоненты (по версии Random Forest):

Цемент — 40-50% важности

Возраст — 15-20% важности

Вода — 8-12% важности

Практическая применимость:
Модель рекомендуется использовать для предварительной оценки прочности бетонной смеси при подборе состава, что позволяет снизить количество дорогостоящих натурных испытаний.

Итоговый вывод
Разработанная модель Random Forest успешно решает задачу прогнозирования прочности бетона с точностью, приемлемой для инженерных расчётов (R² = 0.92, MAE = 4.7 МПа). Анализ подтвердил, что ключевыми факторами, определяющими прочность, являются содержание цемента, возраст образца и водоцементное отношение.
